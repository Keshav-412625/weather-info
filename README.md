# Weather-Cast Web Application

## Project Overview
Weather-Cast is a responsive web application that provides real-time weather information for any city using a weather API. The application is developed using HTML5, CSS3, and JavaScript. It demonstrates the use of Asynchronous JavaScript, Fetch API, JSON, DOM Manipulation, and proper error handling to deliver accurate weather information in a user-friendly interface.

## Features
- Search weather by city name
- Display current temperature
- Show weather description
- Display humidity level
- Show wind speed
- Display weather icon
- Responsive design for all devices
- Loading indicator while fetching data
- Error handling for invalid city names or network issues

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- JSON
- OpenWeather API

## Installation & Usage
1. Download or clone the project.
2. Obtain a free API key from OpenWeather.
3. Open the `script.js` file and replace the existing API key with your own.
4. Save the file.
5. Open `index.html` in any modern web browser.
6. Enter the desired city name and click the **Search** button to view live weather information.
